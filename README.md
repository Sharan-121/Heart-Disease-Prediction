# Heart-Disease-Prediction
Predicting whether a person has Heart Disease or not through Machine Learning techniques.

# Problem Definition
* Given a Medical Dataset,predict whether the person has Heart Disease or not.
* The given Dataset comprises of 14 columns including the Target variable.
* The Target variable is 0 or 1 indicating whether the person is diagnosed with Heart Disease or not.

# Data
* This Dataset is available in Kaggle.</br>
* Please refer this link for downloading the dataset: (https://www.kaggle.com/ronitf/heart-disease-uci)

# Evaluation
* Three models(KNN,Random Forest,Logistic Regressor) were trained and tested on the Dataset.
* After Hyperparameter Tuning using GridSearchCV,Logistic Regressor model produced an accuracy of about 88 percentage.
